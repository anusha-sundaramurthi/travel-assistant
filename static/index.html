❓ How to Use

📄 Uploading PDF Guides

  • Click Upload Guides in the sidebar or drag PDFs onto it.
  • Click ⚡ Process Guides — wait for the ✅ toast.

💬 Asking Questions

  • Type your question in the chat box and hit Enter (Shift+Enter for newline).
  • Ask follow-ups like "what about food there?" — the AI understands context!

💾 PDF Memory

  • Uploaded PDFs are stored permanently in Qdrant — no need to re-upload next session.
  • Use 🗄️ Clear DB only when you want to wipe all stored PDFs.
